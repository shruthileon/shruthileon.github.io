Project Overview

A self-collected dataset turned into a boba analytics tool. I personally visited each cafe, tracking prices, quality, and study conditions - because good data (and boba) takes effort.

Technologies Used

  • Python
  • Pandas
  • Matplotlib
  • Google Sheets API

Project Details

Developed a scoring system to evaluate boba shops based on price, quality, study ambiance, and proximity to UCI.

Automated data collection using Python scripts and Google Sheets API.

Currently working on adding a predictive recommendation model to suggest new study spots.
