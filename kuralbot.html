Project Overview

KuralBot is an AI-powered chatbot that generates responses based on Thirukkural (classical Tamil literature), bridging ancient wisdom with modern AI.

Technologies Used

  • Python
  • Natural Language Processing (NLP)
  • GPT-based Transformer Model
  • Jupyter Notebook

Project Details

Helped developed a chatbot trained on Tamil literary texts using deep learning techniques to generate meaningful and culturally rich responses.

Enhanced multilingual support by optimizing translation APIs, significantly improving response quality.
