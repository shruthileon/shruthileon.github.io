Project Overview

KuralBot is an AI-powered chatbot that generates responses based on Thirukkural (classical Tamil literature), providing ethical and philosophical guidance.

Technologies Used

  • Python
  • Natural Language Processing (NLP)
  • GPT-based Transformer Model
  • Flask

Project Details

KuralBot was trained on Tamil literary texts and uses deep learning techniques to generate insightful and meaningful responses.
