Project Overview

Landmark is an AI-powered tool that recognizes historical and cultural landmarks in real-time using image processing and machine learning.

Technologies Used

  • Python
  • TensorFlow/Keras
  • OpenCV
  • Flask (for API)

Project Details

Developed a deep learning model trained on landmark datasets to classify and provide information on detected sites.
